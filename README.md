## Requirements

- [Node.js](https://nodejs.org/en/)

## Installation

### Cloning this repo

```cmd
> git clone https://github.com/pruizlezcano/where-to-watch.git
> cd where-to-watch
```

### Installing dependencies

```cmd
> npm install
```

### Running the app

```cmd
> npm start
```
